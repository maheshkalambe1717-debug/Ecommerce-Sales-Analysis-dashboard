# Ecommerce-Sales-Analysis-dashboard
Built an interactive E-Commerce Sales Dashboard in Excel  to analyse sales, profit, orders, and regional performance. Implemented KPIs, category analysis, map visualization, and dynamic filters to generate business insights. 
Project Overview
•	Imported and cleaned raw sales data using Excel
•	 Designed interactive dashboard in Power BI
•	 Created KPIs: Total Sales, Profit, Quantity, Orders, Profit Margin
•	Implemented Year-over-Year growth analysis
•	Built category-wise and sub-category-wise sales analysis
•	Designed state-wise sales map visualization
•	Added dynamic filters (Year, Segment, Region) for user interaction
•	Identified top performing products and categories
•	Used charts, slicers, and drill-down features for better insights
